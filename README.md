# 🧠 Smart Converter and Bitwise Analyzer (MIPS Assembly)

## 📌 Description
This is a menu-driven console application written in MIPS Assembly that performs number conversion and bitwise operations.

The program allows users to convert between decimal and binary, analyze binary numbers, and perform bit shifting operations with proper input validation.

---

## 🚀 Features
- 🔢 Convert Decimal to Binary
- 🔁 Convert Binary to Decimal
- 🔍 Count number of 1s and 0s in a binary number
- ↔ Perform left and right bit shifting
- ✅ Input validation for both decimal and binary inputs
- 📋 Menu-driven user interface

---

## 🛠 Technologies Used
- MIPS Assembly Language
- MARS / QtSPIM Simulator

---

## 🧠 Concepts Applied
- Assembly Programming Fundamentals
- Register Management
- Functions using `jal` and `jr`
- String to Integer Conversion
- Bitwise Operations (`sllv`, `srlv`)
- Looping and Branching
- Input Validation (ASCII checking)

---

## 🎯 Purpose
This project was developed to strengthen understanding of low-level programming, data representation, and bitwise operations using MIPS assembly language.
